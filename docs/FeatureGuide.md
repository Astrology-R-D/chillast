# Myst 功能文档 · Feature Guide

本文件详述 Myst 的每一项功能、操作流程、参数含义与星盘计算原理。

---

## 1. 档案管理 Profiles

档案是所有星盘的输入。前往侧边栏「档案管理」。

### 1.1 字段说明

| 字段 | 说明 | 校验 |
| --- | --- | --- |
| 中文名字 | 档案中文名 | 中/英文名至少填一个 |
| 英文名字 | 档案英文名 | 同上 |
| 性别 | 男 / 女 / 其他 | 必选 |
| 出生日期与时间 | **公历**，年/月/日/时/分，精确到分钟 | 合法日历日期，24 小时制 |
| 出生地 | 城市名 + 经纬度 | 纬度 −90~90，经度 −180~180 |
| 备注 | 自由文本 | 可选 |

### 1.2 出生地与时区

- 在「出生地」输入框中检索内置城市（支持中文、英文、国家代码），选中后自动填入经纬度。
- 也可手动输入经纬度，用于内置列表之外的地点。
- **时区与历史夏令时（DST）由出生地坐标自动推算**，无需手动选择时区。例如 1990 年的北京会正确应用当年的夏令时规则。

### 1.3 操作

- **新建档案**：点击「＋ 新建档案」，填写后保存。
- **编辑 / 删除**：在档案卡片右侧操作。
- **选为主体**：点击卡片可将其设为后续星盘的默认主体。
- **持久化**：所有档案以 JSON 写入本机用户数据目录（原子写入，防止崩溃损坏）。

## 2. 个人星盘 Personal charts

前往「个人星盘」，选择档案、星盘类型与参数，点击「生成星盘」。

### 2.1 本命盘 Natal（本命盘）
出生时刻的星图，呈现行星落座、宫位与盘内相位。这是其它所有推运的基础。

### 2.2 行运盘 Transit（行运盘）
将「目标日期」的天体叠加在本命盘外环，内环为本命、外环为行运，相位列表为
**行运对本命**的交角。用于观察当前天象对个人的影响。
- 参数：**目标日期与时间**（默认当前时刻）。

### 2.3 二次推运 Secondary Progressions（二次推运）
象征推运法「一天等于一年」：出生后第 N 天的星图代表生命第 N 年。
内环本命、外环推运，列出两者交角。
- 参数：**目标日期**（据此计算已历年数）。

### 2.4 太阳返照 Solar Return（太阳返照）
求指定年份中**行运太阳精确回到本命太阳黄经**的时刻，并以该时刻起一张完整星盘。
程序在生日前后 ±5 天窗口内用根查找（夹逼 + 二分）定位精确返照时刻。
- 参数：**返照年份**。

### 2.5 月亮返照 Lunar Return（月亮返照）
求**行运月亮回到本命月亮黄经**的时刻（周期约 27.32 天），取距目标日期最近的一次。
- 参数：**目标日期**（在其 ±14 天窗口内定位）。

## 3. 合盘分析 Relationship charts

前往「合盘分析」，选择**主体（内环）**与**次体（外环）**两个不同档案。

### 3.1 比较盘 Synastry（比较盘）
两张本命盘叠加：内环为主体、外环为次体，相位列表为**双方天体之间的交角**，
宫位采用主体本命盘。用于分析两人之间的互动。

### 3.2 组合中点盘 Composite（组合中点盘）
对两盘的每个天体、四轴与宫头取**圆周中点**，合成一张象征「关系本身」的星盘。
由于组合盘没有真实时刻，宫位由中点宫头几何推算，天体按几何落宫。

### 3.3 戴维森盘 Davison（戴维森时空中点盘）
取两人出生**时间的中点**与**地理坐标的中点**，起一张**真实**星盘——
因此它拥有真实的宫位与行星运动（含逆行）。

## 5. 星盘界面解读 Reading the chart

### 5.1 星盘轮 Wheel
- **最外圈**：黄道十二宫，按元素着色（火/土/风/水），含 5° 刻度。
- **四轴**：上升 Asc、天顶 MC、下降 Dsc、天底 IC，以金色标注。
- **宫位线**：从中心放射至黄道，角宫（1/4/7/10）线更粗；宫号位于宫弧中点。
- **行星**：以符号绘制，附座内度数与逆行标记 `℞`；密集处自动展开避让，并以引线指向真实位置。
- **相位连线**：中心区域连接成相位的天体。

### 5.2 相位配色 Aspect colours
| 颜色 | 含义 |
| --- | --- |
| 绿色 | 柔和相位（三分、六分） |
| 红色 | 紧张相位（四分、对分） |
| 金色 | 合相 / 四轴 |
| 灰色虚线 | 次要相位（梅花、八分、十二分等） |

连线不透明度随相位精确度（容许度越小越实）变化。

### 5.3 数据面板 Panels
- **行星位置**：符号、名称、座内度数、星座、宫位（双环盘分环列出）。
- **元素与模式分布**：以主体本命天体统计火/土/风/水与基本/固定/变动比例。
- **相位**：按容许度升序列出所有相位及其类型与度数。

### 5.4 导出 Export
点击星盘右上「导出 SVG」可将当前星盘保存为矢量图。

## 7. 常见问题 FAQ

**Q：出生时间不准会怎样？**
上升点与宫位对时间高度敏感；几分钟误差即可能改变上升星座。请尽量使用精确到分的真太阳出生时间。

**Q：找不到出生城市？**
直接在经纬度框手动输入坐标即可，时区会据坐标自动推算。

**Q：数据保存在哪里？**
`<userData>/data/Profiles.json`（Windows 约为 `%APPDATA%/Myst/data/`）。
